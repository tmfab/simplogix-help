# Defects

Defects are issues found with assets — anything from minor wear to safety-critical faults. SIMPLOGIX helps you log, track, and resolve defects systematically.

## Defect statuses

| Status | Meaning |
|---|---|
| **Open** | Defect has been logged and is awaiting action |
| **In progress** | Work is underway to resolve the defect |
| **Resolved** | The defect has been fixed and closed |

## The defects list

The defects page shows all defects for your current site. Use the filters to narrow by status, asset, severity, or date.
