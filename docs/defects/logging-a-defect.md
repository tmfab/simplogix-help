# Logging a Defect

Admins, Managers, and Operators can log defects.

## From the defects page

1. Go to **Defects** in the sidebar
2. Click **Log defect**
3. Select the **asset** the defect relates to
4. Fill in the details:
   - **Title** — a brief description of the issue
   - **Severity** — low, medium, high, or critical
   - **Description** — full details of what was found
   - **Photos** — attach photos if available
5. Click **Submit**

## From an inspection

When running an inspection, you can log a defect directly against a failed checklist item. This automatically links the defect to the inspection for traceability.

## After logging

The defect will appear in the defects list with a status of **Open** and can be assigned to a team member for resolution.
