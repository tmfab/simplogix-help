# Company Settings

The **Company Settings** area is where Administrators manage their company's configuration, users, sites, and templates. Access it from the profile menu in the top-right corner.

## Sections

| Section | What it's for |
|---|---|
| **General** | Company branding (logo, colours), localisation (date/time formats, timezone, currency), and working hours |
| **Sites** | Create, edit, clone, and archive sites. Manage plant area hierarchies and per-site user access |
| **Users** | Invite new users, manage roles and site access, resend invitations, deactivate accounts |
| **Templates** | Configure inspection checklists, defect forms, asset forms, and shared lookup lists |
| **Documents** | Upload and version company-wide documents (e.g. procurement specifications) |
| **Import / Export** | Bulk import data from CSV, export data on-demand or on a schedule |
| **Logs & Audits** | View system activity logs, security alerts, notification delivery logs, and data change history |
| **Notifications** | Configure default notification preferences per role, escalation rules, email digests, and quiet hours |

## Who can access Company Settings?

Only users with the **Admin** role can access Company Settings. Managers can invite users (up to their own role level) but cannot access the full settings area.
