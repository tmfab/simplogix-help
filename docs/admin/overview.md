# Company Settings

**Company Settings** is where Administrators manage the company's configuration, users, sites, and templates. Access it from the profile menu in the top-right corner.

## Sections

| Section | What it's for |
|---|---|
| **General** | Company branding (logo, colours), localisation (date/time formats, timezone, currency), and working hours |
| **Site Management** | Creating, editing, cloning, and archiving sites. Plant area hierarchies and per-site user access |
| **Users & Permissions** | Inviting new users, managing roles and site access, resending invitations, deactivating accounts |
| **Templates** | Inspection checklists, defect forms, asset forms, and shared lookup lists |
| **Company Documents** | Uploading and versioning company-wide documents (e.g. procurement specifications) |
| **Import / Export** | Bulk importing from CSV, on-demand or scheduled exports |
| **Logs & Audits** | System activity logs, security alerts, notification delivery logs, and data change history |
| **Notifications** | Default notification preferences per role, escalation rules, email digests, and quiet hours |

## Who can access Company Settings?

Only users with the **Admin** role. Managers can invite users (up to their own role level) but can't access the full settings area.
