# Import and Export

Go to **Company Settings > Import / Export** to manage bulk data operations.

## Importing data

### Quick import

1. Go to the **Import Data** tab
2. Upload a CSV file using the drag-and-drop zone
3. Map your CSV columns to SIMPLOGIX fields
4. Review the validation preview -- the system highlights any errors or warnings
5. Confirm and import

### Import templates

Save your column mapping configuration as an **import template** for reuse. This is useful when you regularly import data in the same format.

## Exporting data

### Quick export

1. Go to the **Quick Export** tab
2. Select what to export: Defects, Assets, Inspections, Work Orders, Audit Logs, Users, or Sites
3. Choose a format: CSV, JSON, or XLSX
4. Optionally filter by site and/or date range
5. Select which columns to include
6. Click **Export Now**

### Export templates

Save your export configuration (columns, filters, format) as a template for repeat use.

### Scheduled exports

Set up recurring exports that run automatically:

- Choose frequency: daily, weekly, or monthly
- Configure the export settings (data type, columns, filters, format)
- Exports run on schedule and are available for download

## History

The **History** tab shows a log of all past import and export jobs with their status (pending, processing, complete, failed, or cancelled).
