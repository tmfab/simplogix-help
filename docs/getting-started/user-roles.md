# User Roles

SIMPLOGIX uses role-based access control. Your role determines what you can see and do within the app.

## Role overview

| Role | Description |
|---|---|
| **Admin** | Full access — manages users, site settings, assets, inspections, and defects |
| **Manager** | Can view all data and manage inspections and defects, but cannot manage users or site settings |
| **Operator** | Can run inspections and log defects, but cannot modify site configuration |
| **Viewer** | Read-only access to assets, inspections, and defect reports |

## Permissions at a glance

| Action | Admin | Manager | Operator | Viewer |
|---|:---:|:---:|:---:|:---:|
| View assets | ✓ | ✓ | ✓ | ✓ |
| Add / edit assets | ✓ | ✓ | | |
| Run inspections | ✓ | ✓ | ✓ | |
| View inspections | ✓ | ✓ | ✓ | ✓ |
| Log defects | ✓ | ✓ | ✓ | |
| Manage defects | ✓ | ✓ | | |
| Manage users | ✓ | | | |
| Site settings | ✓ | | | |

## Changing a user's role

Only **Admins** can change user roles. Go to **Settings → Users**, find the user, and update their role from the dropdown.
