# The Dashboard

After logging in, you land on the **Dashboard** — a real-time overview of your site's current status.

## What you'll see

### Summary cards

At the top of the dashboard you'll find summary cards showing:

- **Open defects** — total number of unresolved defects on your site
- **Inspections this month** — how many inspections have been completed in the current month
- **Assets** — total number of active assets registered to your site

### Recent activity

Below the summary cards, the dashboard shows recent activity across your site — newly logged defects, completed inspections, and any critical issues that need attention.

## Acting from the dashboard

Click any summary card or activity item to jump straight to the relevant record. For example, clicking **Open defects** takes you to the defects list filtered to open items.

## Site selector

If you have access to more than one site, use the **site selector** in the top navigation to switch between sites. The dashboard updates to reflect the selected site.
