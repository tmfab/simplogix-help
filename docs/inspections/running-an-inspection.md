# Running an Inspection

Admins, Managers, and Operators can run inspections.

## Steps

1. Go to **Inspections** in the sidebar
2. Click **New inspection**
3. Select the **asset** you are inspecting
4. Work through the inspection checklist — mark each item as pass, fail, or not applicable
5. Add any **notes** relevant to the inspection
6. If you find an issue, you can **log a defect** directly from the inspection
7. Click **Submit** to save the inspection

## Submitting with open items

If you cannot complete all checklist items (e.g. you need to return later), you can save the inspection as a **draft** and come back to it. Drafts are visible only to you until submitted.

## After submitting

Once submitted, the inspection is added to the asset's history and cannot be edited. If you made an error, contact your site administrator.
