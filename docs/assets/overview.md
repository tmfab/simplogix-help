# Assets

The **Assets** section is your site's centralised register of equipment, infrastructure, and anything else you need to track, inspect, or maintain.

## What is an asset?

An asset is any physical item you want to inspect or monitor -- a pump, a vehicle, a piece of electrical equipment, a structure, or any other item relevant to your operations.

## The asset list

The assets page shows all assets registered to your current site. You can view your assets in four different ways:

| View | Description |
|---|---|
| **List** | A sortable, configurable data table -- customise which columns are visible |
| **Grid** | Card-based view showing asset details at a glance |
| **Tree** | Hierarchical view grouped by plant area -- expand an asset to see its recent inspections and defects |
| **Map** | Geographic view showing assets on a map based on geotagged photo locations |

## Searching and filtering

- **Search** by asset name, ID, or code using the search bar -- results appear as you type
- **Filter** by status (Active, Inactive, Maintenance, Decommissioned), criticality (High, Medium, Low), asset type, plant area, and inspection dates
- Use **filter presets** for common views like "High Criticality" or "Needs Maintenance"
- Active filters appear as dismissible chips below the toolbar

## Bulk operations

Managers and Admins can select multiple assets in list view to:

- Update status in bulk
- Delete multiple assets
- Export selected assets to CSV

## Importing and exporting

Use the toolbar dropdown to **import assets from CSV** or **export** your current view (all or selected) to CSV.
