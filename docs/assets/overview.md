# Assets

The **Assets** section is your site's centralised register — equipment, infrastructure, vehicles, and anything else you need to track, inspect, or maintain.

## What is an asset?

An asset is any physical item you want to inspect or monitor. That could be a pump, a vehicle, a piece of electrical equipment, a structure, or anything else relevant to your operations.

## The asset list

The assets page shows all assets registered to your current site. There are four views:

| View | Description |
|---|---|
| **List** | Sortable data table with configurable columns |
| **Grid** | Card-based view showing key details at a glance |
| **Tree** | Hierarchical view grouped by plant area — expand an asset to see its recent inspections and defects |
| **Map** | Geographic view showing assets on a map based on geotagged photo locations |

## Searching and filtering

- **Search** by asset name, ID, or code — results appear as you type
- **Filter** by status (Active, Inactive, Maintenance, Decommissioned), criticality, asset type, plant area, and inspection dates
- **Filter presets** give you quick access to common views like "High Criticality" or "Needs Maintenance"
- Active filters show as dismissible chips below the toolbar

## Bulk operations

Managers and Admins can select multiple assets in list view to update status, delete, or export to CSV.

## Importing and exporting

Use the toolbar dropdown to import assets from CSV or export your current view to CSV.
